# Data Directory Guide

This directory documents local data organization for reproducibility and evaluation.

## Current policy

- No scientific dataset is fabricated in this repository.
- Any externally published dataset/DOI must be added only after formal author curation.

## Structure

- `data/demo/` — demo/scaffold material for local development and documentation.

## Notes for SoftwareX preparation

- If a validated demo dataset is approved, place it under `data/demo/`.
- Record provenance, license, and schema before release/publication.
