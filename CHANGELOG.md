- Consolidated software citation metadata at repository root (`CITATION.cff`) and `codemeta.json`.

### Notes
- This changelog entry focuses on editorial/metadata alignment for release **1.0.0** and SoftwareX submission preparation; no functional runtime behavior was intentionally changed.
