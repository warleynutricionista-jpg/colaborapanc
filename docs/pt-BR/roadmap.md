# Roadmap

## Linha de base atual (implementado)

- Fluxo científico ponta a ponta com inferência/revisão/validação.
- Endpoints operacionais de monitoramento de integrações.
- APIs de paridade mobile para identificação e metadados offline.
- Módulos ambientais para MapBiomas e alertas climáticos.

## Prioridades de curto prazo

- Melhorar observabilidade de falhas/retries nas integrações.
- Expandir cobertura de testes para permissões e edge cases de API.
- Prosseguir limpeza documental e reduzir duplicidade legada.
- Evoluir UX offline no mobile com maior rastreabilidade de pacotes.

## Observação

Detalhes legados de roadmap permanecem disponíveis no histórico de `docs/en/roadmap.md` e em `docs/pt/roadmap.md`.
