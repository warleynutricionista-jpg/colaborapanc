# Guia do Usuário

## Jornada típica

1. Entrar na conta ou criar cadastro.
2. Criar um ponto PANC georreferenciado com imagem/contexto.
3. Consultar o resultado de identificação assistida por IA.
4. Acompanhar ciclo de moderação/validação.
5. Consumir alertas, recomendações e apoio de rotas.

## Destaques no uso mobile

- Previews de mapa e fluxos de ponto.
- Integração com endpoint de identificação por imagem.
- Fluxo de pacotes de plantas offline seletivas.
- Notificações e fluxos de revisão.

## Troubleshooting

Consulte `docs/pt/troubleshooting.md` para problemas conhecidos de setup/execução.


## Operação Mobile em Campo (detalhada)

1. Abra o fluxo de mapa/captura e registre ponto com imagem quando possível.
2. Use sugestões de identificação por imagem como apoio assistivo.
3. Se estiver offline, continue coletando; pendências ficam enfileiradas localmente.
4. Ao reconectar, sincronize pendências e valide o status de sync na interface.
5. Para uso offline seletivo, baixe pacotes de espécies antes da atividade de campo.
6. Lembrete: decisão científica final permanece com revisão/validação humana.

Para comportamento avançado (AR, auto-detecção, pacotes offline, IA assistiva), consulte [`fluxos-mobile-avancados.md`](./fluxos-mobile-avancados.md).
