#Roadmap

## Ready features
- Registration and visualization of georeferenced PANC points.
- Basic scientific flow: AI inference, review queue, validation and history.
- Scientific dashboard with key operational metrics.
- Advanced point search and plant listing.
- Notifications/messages/routes/preferences APIs (basic functional level).

## Partial features
- End-to-end push notifications.
- Selective offline with mature synchronization.
- AR with full pipeline of assets/production experience.
- Automated recommendations with quality governance.
- Consolidation of mobile navigation (React Navigation vs expo-router).

## Future features
- Advanced scientific metrics (F1, confusion matrix per domain).
- Formal versioning of the API and stable contracts per version.
- Advanced geographic privacy policies by type of audience.
- Evolution of the backend architectural modularization.
