# Citation Guidance (Pre-DOI)

## Current status

ColaboraPANC currently provides local citation metadata files:

- `CITATION.cff` (repository root)
- `codemeta.json` (repository root)

No DOI/PID is declared in this repository at this stage.

## How to cite before DOI assignment

Use the software name, version, repository URL, and access date according to your target style guide.

Suggested elements:

1. Software name: **ColaboraPANC**
2. Version: **1.0.0** (or the version/tag used)
3. Repository URL: `https://github.com/warleynutricionista-jpg/colaborapanc`
4. Access date: `[TO_FILL_BY_AUTHORS]`

## Mandatory author-maintainer confirmation before submission

The following fields remain intentionally unresolved and must be confirmed by the project authors/maintainers:

- Final author list
- Affiliations
- ORCID identifiers (if available)
- Corresponding contact e-mail
- DOI/PID (when generated externally)
