# FAQ

## Is ColaboraPANC only a web app?
No. It has a Django web/API backend and an Expo/React Native mobile app.

## Is AI fully autonomous in species validation?
No. The project uses AI-assisted inference with human review and validation workflows.

## Where do I find the full API list?
`mapping/urls.py` is the source of truth. See `docs/en/api.md` for grouped navigation.

## Is there Portuguese documentation?
Yes: `docs/pt-BR/` is the canonical Portuguese tree.

## What about legacy docs in `docs/pt/` and old filenames in `docs/en/`?
They are retained for historical traceability. The canonical curated set is now `docs/en/` and `docs/pt-BR/` indexes.
