# Política de dados e privacidade (base técnica)

## Dados tratados
- Cadastro de usuário (conta/autenticação).
- Dados de observação de ponto (local, relato, foto).
- Metadados de inferência e validação científica.
- Dados de interação (mensagens, notificações, gamificação).

## Dados sensíveis no contexto
- Geolocalização pode revelar áreas ambientalmente sensíveis.
- Imagens podem conter metadados e informações contextuais.

## Diretrizes LGPD aplicáveis
- Finalidade: mapeamento e validação de PANC.
- Necessidade: coletar mínimo necessário para operação.
- Transparência: informar que IA é assistiva e sujeita a erro.
- Segurança: segredos fora do código, controle de acesso por papel.

## Retenção e minimização
- Manter histórico científico para auditabilidade.
- Avaliar política de retenção por tipo de dado (conta, imagem, log).
- Aplicar anonimização/geoprecisão reduzida em contextos públicos sensíveis (pendente de política operacional completa).
