# Roadmap

## Funcionalidades prontas
- Cadastro e visualização de pontos PANC georreferenciados.
- Fluxo científico básico: inferência IA, fila de revisão, validação e histórico.
- Dashboard científico com métricas operacionais principais.
- Busca avançada de pontos e listagem de plantas.
- APIs de notificações/mensagens/rotas/preferências (nível funcional básico).

## Funcionalidades parciais
- Push notifications ponta a ponta.
- Offline seletivo com sincronização madura.
- AR com pipeline completo de ativos/experiência em produção.
- Recomendações automatizadas com governança de qualidade.
- Consolidação de navegação mobile (React Navigation vs expo-router).

## Funcionalidades futuras
- Métricas científicas avançadas (F1, matriz de confusão por domínio).
- Versionamento formal da API e contratos estáveis por versão.
- Políticas avançadas de privacidade geográfica por tipo de público.
- Evolução da modularização arquitetural do backend.
